# 🍅 FocusFlow

A modern Pomodoro Timer built using **HTML, CSS, and JavaScript** to help users stay focused and manage their study or work sessions effectively.

## 🌐 Live Demo

🔗 https://divyanshisr1060-cmd.github.io/FocusFlow/

---

## ✨ Features

- ⏱️ 25-minute Pomodoro timer
- ▶️ Start timer
- ⏸️ Pause timer
- 🔄 Reset timer
- 🎨 Clean and modern user interface
- 📱 Responsive design

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript

---

FocusFlow/

## 🚀 How to Run Locally

1. Clone this repository
2. Open the project folder
3. Open `index.html` in your browser

Or visit the live demo above.
